Politique de Confidentialité

1. Introduction

​Bienvenue sur The World ! La protection de vos données personnelles est une priorité pour nous. Cette politique de confidentialité vous explique quelles informations nous collectons, comment nous les utilisons et quelles sont vos options concernant ces informations.

​

2. Données que nous collectons

​Notre site ne collecte pas de données personnelles directement identifiables, telles que votre nom, adresse e-mail ou numéro de téléphone, car nous ne proposons ni inscription ni formulaire (sauf dans le cas d'un formulaire de contact).

​Cependant, comme la plupart des sites web, nous utilisons des outils d'analyse (comme Google Analytics, par exemple) pour collecter des données de navigation non personnelles. Ces données incluent :

​
  • Votre type d'appareil (ordinateur, mobile, etc.)
    • ​
  • Votre navigateur web

  • ​Les pages que vous visitez sur notre site

  • ​Le temps que vous passez sur ces pages

  • ​Votre localisation approximative (pays, ville)

Ces informations nous aident à comprendre comment les visiteurs utilisent notre site afin de l'améliorer.


​3. Utilisation de vos données

​Les données de navigation collectées sont utilisées exclusivement pour :

  • ​Améliorer l'expérience utilisateur : Savoir quelles pages sont les plus populaires nous aide à optimiser le contenu et la navigation.
  • ​Maintenir la sécurité du site : Analyser le trafic nous permet de détecter et de prévenir d'éventuelles menaces.

​4. Vos droits

Puisque nous ne collectons aucune donnée personnelle vous identifiant directement, vous n'avez pas d'actions spécifiques à prendre. Néanmoins, sachez que vous pouvez configurer votre navigateur pour refuser les cookies ou être alerté lorsqu'un cookie est envoyé. Cela peut toutefois affecter le bon fonctionnement de certains sites.


​5. Contact

Si vous avez des questions concernant cette politique de confidentialité, vous pouvez nous contacter via la page "Nous contacter" de notre site.
